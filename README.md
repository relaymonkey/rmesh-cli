# rmesh — RelayMesh CLI

Public, Apache-2.0 command-line tool for RelayMesh — authenticate, manage networks, inspect traffic, and operate local nodes from the terminal.

## Quick start

```bash
rmesh auth login
rmesh network list
rmesh network use <slug>
rmesh traffic text
```

## What it does

- Sign in and manage which network commands target by default
- List, filter, and stream mesh traffic from the terminal
- Connect local nodes and forward traffic to RelayMesh
